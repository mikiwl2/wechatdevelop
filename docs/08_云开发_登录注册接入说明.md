# 云开发登录与资料同步说明

## 1. 当前已接通能力

云环境：`cloud1-0g15dqxu9ad791a6`
云函数：`auth_v2`
用户集合：`users`

已支持云函数动作：

- `action=login`
- `action=syncProfile`
- `action=getProfile`
- `action=saveProfile`

## 2. 前端调用位置

- 登录页（我的）：`pages/mine/mine.js`
  - 登录时调用 `login`
  - 首次授权头像昵称时调用 `syncProfile`
- 资料页：`pages/edit-profile/edit-profile.js`
  - 进入页面时调用 `getProfile`
  - 保存资料时调用 `saveProfile`

## 3. 你需要执行的部署步骤

1. 在微信开发者工具选择环境 `cloud1-0g15dqxu9ad791a6`
2. 确认数据库集合 `users` 已创建
3. 右键 `cloudfunctions/auth_v2` -> 上传并部署（云端安装依赖）
4. 重新编译小程序

## 4. 验证步骤

1. 登录后打开资料页，修改昵称/头像/身材字段并保存
2. 退出再登录，资料应自动回显
3. 更换微信号登录，应看到不同账号资料

## 5. 常见报错处理

- `ResourceNotFound.Function`
  - 说明云端没有该函数，请先创建或直接上传部署 `auth_v2`
- `ResourceInUse.FunctionName`
  - 说明同名函数已存在，请改为“更新部署”或使用新函数名
- 登录提示 cloud function failed
  - 检查函数日志、集合权限和环境 ID
