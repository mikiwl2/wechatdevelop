# 测试验收清单

更新时间：2026-03-08

## 1. 冒烟用例（每次提测必过）
1. 启动后进入“我的”，未登录仅见登录入口。
2. 未登录点击“AI试衣/AI衣橱”会跳回“我的”并提示登录。
3. 登录成功后可访问三大页面全部功能。

## 2. 登录与账号隔离
1. A账号登录生成作品 -> 退出 -> A账号再登录：作品与资料恢复。
2. A账号登录后退出 -> B账号登录：不应看到A账号作品。
3. 后续登录不应重复出现“是否使用微信资料”弹窗。

## 3. 试衣流程
1. 未上传双图点击生成：提示缺少素材。
2. 上传双图后生成成功弹窗出现。
3. 仅保存：作品出现在“我的试衣”，不进入衣橱公开流。
4. 发布并保存：作品出现在“我的试衣”和“AI衣橱”。

## 4. 互动流程
1. 衣橱页点赞/收藏可切换并即时回显。
2. 详情页点赞/收藏与列表表现一致。
3. 评论可发布并刷新显示。

## 5. 我的页与资料页
1. 资料页修改昵称头像后，“我的”页即时更新。
2. 删除与批量删除后，列表与统计一致变化。
3. 悬浮球只显示“+ 生成更多”。

## 6. 编译与兼容检查
1. 全项目无 BOM 导致的 wxss/wxml 编译报错。
2. 开发者工具基础库 `3.14.3` 下可正常编译运行。
3. Skyline/WebView 配置一致，不出现无效警告。

## 7. 发布前检查项
1. 文案检查：统一“AI试衣/AI衣橱/我的”。
2. 资源检查：默认头像、占位图、Tab 图标可加载。
3. 异常检查：空列表、作品不存在、评论空输入均有合理提示。
